How to Use WeatherWise

Finding Your Location

There are three ways to select a location on WeatherWise:

  1. Search by name: Type a city or location name in the search box and press Enter.
  2. Click on the map: Simply click anywhere on the map to select that location.
  3. Drop a pin: Use the pin tool to precisely mark a location of interest.

Once selected, the latitude and longitude coordinates will be displayed, and you can proceed with your weather analysis.

Selecting Time Period

WeatherWise uses historical NASA data to predict weather conditions:

  • Use the month selector to choose which time of year you're interested in.
  • The analysis will be based on historical data for that month across multiple years.
  • This gives you a statistical likelihood of weather conditions, not a forecast.

Note: WeatherWise is designed for planning purposes, not for current weather forecasts.

Choosing Weather Variables

Select which weather conditions you want to analyze:

  • Temperature: How hot or cold it's likely to be
  • Precipitation: Likelihood of rain or snow
  • Wind Speed: How windy conditions might be
  • Humidity: Expected moisture levels in the air

For each variable, WeatherWise will calculate the probability of "extreme" or "uncomfortable" conditions based on historical data.

You can adjust what counts as "extreme" in the Settings page.

Understanding Results

After analysis, WeatherWise displays results in several formats:

  • Text Summary: A plain-language description of likely conditions
  • Probability Charts: Visual representation of how likely different conditions are
  • Map Overlays: Color-coded visualization of conditions in the selected area

The results show statistical probabilities based on historical data, not predictions of actual future weather.

Exporting Data

You can download the analysis results for your records or further analysis:

  1. Click the "Download Data" button in the results section
  2. Choose CSV or JSON format
  3. The downloaded file will include all analyzed data, metadata, and source information

This feature is useful for researchers, travelers planning trips, or anyone who needs offline access to the analysis.

WeatherWise requires an internet connection to fetch data from NASA APIs. However, once you've performed an analysis, you can download the results as CSV or JSON files for offline reference. The website itself cannot function without an internet connection for the initial data retrieval.

About the Data

WeatherWise uses data from the following NASA sources:

  • NASA POWER API: Provides surface meteorology and solar energy data
  • MERRA-2: Modern-Era Retrospective analysis for Research and Applications, Version 2
  • GPM: Global Precipitation Measurement mission data

The data is processed client-side to calculate probabilities and generate visualizations. No user data is stored on servers.

For more information about NASA's data sources, visit NASA POWER.
